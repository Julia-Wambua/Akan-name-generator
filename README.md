## Live site
https://julia-wambua.github.io/Akan-name-generator/

## Description
This is a name generator website that uses user input (birthdate and gender) to generate an akan name,which is a ghanian name based on the day of the week one was born and their gender,for the user.
This project aims at showcasing aquired skills such as:
  1. Structuring a website using HTML
  2. Styling a website using CSS
  3. Using JavaScript to make the website user interactive.
  4. Using Git and GitHub for version control
  5. Deploying a live site using GitHub Pages
     
## Setup/Installation Requirements
* Download the repository from GitHub
* Ensure the presence of index.html, style.css and an image used in creation of the website.
* Click on the live site on the repository.
